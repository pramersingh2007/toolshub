# **App Name**: ToolForge

## Core Features:

- Dynamic Homepage Content Display: Renders the hero section (headline, subheading, prominent search bar, CTA), tool categories section with clickable cards and icons, popular tools showcase, a 'Why Choose Us' value proposition section, and testimonials, all designed responsively for optimal viewing.
- AI Tool Recommendation & Search: A prominent search bar feature acting as an AI tool to help users quickly find suitable tools, capable of interpreting natural language queries and suggesting relevant tools or tool categories, enhancing user discovery and engagement.
- Optimized Ad Integration: Strategic and non-intrusive placement of various ad formats including banners, in-content ads, and a sticky sidebar ad, carefully positioned to maximize ad revenue without disrupting the user experience.
- Dark Mode Toggle: A user-selectable UI switch that allows users to seamlessly change the entire application interface between a light and a dark color scheme for personalized viewing preferences and reduced eye strain.
- SEO & Performance Optimization: Core technical implementation focused on achieving fast loading times (optimized images, lazy loading), full responsiveness for a mobile-first experience, SEO-friendly HTML structure (H1, meta tags), and accessibility standards (ARIA labels).

## Style Guidelines:

- Color scheme reflects reliability and efficiency with a modern touch. Primary blue: #1C74C9 (HSL: 210, 75%, 45%) for actionable elements and brand emphasis. Background color: #EBF3F9 (HSL: 210, 15%, 95%) provides a clean, very light foundation with a hint of the primary hue. Accent color: #4D4DB8 (HSL: 240, 60%, 55%) as an analogous deep purple for highlight elements, ensuring good contrast and visual interest.
- Headline and body font: 'Inter', a grotesque sans-serif that offers a modern, neutral, and highly readable aesthetic, suitable for a professional SaaS-like interface. Its versatility ensures a clean and objective look across all content.
- Utilize a consistent set of modern, minimalist line icons for tool categories and features. Icons should be clear, intuitive, and complement the clean UI, appearing subtly animated on hover.
- A clean, card-based grid layout will organize content, providing distinct sections for categories and popular tools. The hero section features a prominent, centered search bar. Strategic use of white space ensures clarity and ease of navigation. Layout will be fully responsive, prioritizing mobile-first adaptability, and includes specific placements for banner ads, in-content ads, and a sticky sidebar ad.
- Subtle and smooth hover animations for interactive elements like cards, buttons, and links will provide user feedback. Loading skeleton UIs will be implemented to enhance perceived performance during content fetching, reducing perceived wait times and improving user experience.
