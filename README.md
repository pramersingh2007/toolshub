# ToolsHub

ToolsHub is a comprehensive suite of online tools including image processing, PDF management, and content generation.

## Features

- **Image Tools**: Compression, resizing, merging, white background addition, and passport photo making.
- **PDF Tools**: Compression and merging.
- **Utility Tools**: QR generator, bio-data maker, word counter, and JSON formatter.
- **AI Integration**: Powered by Google GenAI and Genkit.

## Getting Started

### Prerequisites

- Node.js
- npm/yarn/pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

To start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:9002`.

## Technologies Used

- **Framework**: Next.js 15
- **AI**: Google GenAI, Genkit
- **Styling**: TailwindCSS, Radix UI
- **PDF**: pdf-lib, pdfjs-dist, jspdf
- **Images**: cropperjs, html2canvas
